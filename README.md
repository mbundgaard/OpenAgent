# OpenAgent

Personal AI agent framework. Multi-channel platform connecting LLM providers (text, voice) to inbound channels (REST API, WebSocket, Telegram, webhooks) with a shared agent personality layer.

## Project Board

https://github.com/users/mbundgaard/projects/4

## Tech Stack

- .NET 10, ASP.NET Core Minimal APIs
- SQLite for conversation persistence
- Azure OpenAI for text and voice
